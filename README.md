# BILLIONAIRES CLUB — исправленный фон для компьютера

В этой версии фон сделан отдельным реальным слоем, поэтому он точно отображается:

- центральный макет не растягивается;
- по бокам виден размытый фон из того же дизайна;
- на телефоне ничего не меняется.

Загрузите все файлы из архива в GitHub поверх текущих и нажмите Commit changes.
После обновления откройте сайт через Cmd+Shift+R или в режиме инкогнито.
