# BILLIONAIRES CLUB — бесплатный сайт / Telegram Mini App

## Что уже есть
- адаптивный премиальный интерфейс;
- модули и уроки;
- окно урока;
- место для Kinescope;
- отметка пройденных уроков;
- прогресс сохраняется на устройстве;
- подключён Telegram Web App SDK;
- кнопка перехода в чат.

## Как вставить видео Kinescope
Открой `app.js`.

У каждого урока есть поле:

```js
video: ""
```

Вставь туда ссылку из iframe Kinescope, например:

```js
video: "https://kinescope.io/embed/XXXXXXXX"
```

Используй именно URL из `src="..."` кода встраивания.

## Как указать чат
В `app.js` найди:

```js
https://t.me/PUT_YOUR_CHAT_USERNAME
```

Замени на ссылку своего чата.

## Как бесплатно опубликовать через Vercel
1. Зарегистрируйся на GitHub.
2. Создай новый репозиторий.
3. Загрузи в него `index.html`, `styles.css`, `app.js`.
4. Зайди на Vercel.
5. Нажми Add New → Project.
6. Выбери репозиторий.
7. Нажми Deploy.
8. Получишь HTTPS-ссылку вида `https://...vercel.app`.

## Как подключить к Telegram
1. Создай бота через @BotFather.
2. Открой Bot Settings.
3. Выбери Menu Button или Configure Mini App.
4. Укажи HTTPS-ссылку с Vercel.
5. Назови кнопку «Открыть курс».

## Важно
Сейчас это бесплатный MVP без серверной авторизации. Любой человек со ссылкой сможет открыть сайт.
Для платного курса следующим этапом нужно добавить:
- проверку Telegram ID на сервере;
- базу активных учеников;
- выдачу видео только после проверки доступа.
